# The Researcher's Guide to Peptides - Landing Page

This is the landing page for "The Researcher's Guide to Peptides: Volume 1", a free ebook funnel designed to capture leads and educate researchers on peptide quality and COA validation.

## Tech Stack

- React 19
- TypeScript
- Vite
- Material UI (MUI) v6
- Emotion

## Features

- High-conversion Hero section with 2025 UX best practices
- Lead capture forms (First Name, Last Name, Email, Phone)
- Detailed "What's Inside" Table of Contents
- Responsive design (Mobile & Desktop optimized)
- Clean, authority-focused aesthetic

## Getting Started

1. `npm install`
2. `npm run dev`
